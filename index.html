? How to Play
Gameplay
Tap anywhere on the arena to drop a token. When two identical tokens touch, they merge into the next level token and increase your Ecosystem Value.
Game Over
If any token stays above the red danger zone for 3 seconds, the game ends. Don't let the arena overflow!
Token Evolution
SEED → ORYN → ROVA → CNOTE → AURELIS → TERROVA → BLACKSITE → ORYNTH CORE
Tips
• Stack same tokens vertically for easy merges
• Larger tokens are worth more points
• Unlock all 8 tokens to complete the Collection
